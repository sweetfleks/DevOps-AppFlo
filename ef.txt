Обновление списка пакетов: Перед установкой пакета, обновите список доступных пакетов с помощью команды:

sudo urpmi --auto-update

Bash
Установка openssh-server: Теперь установите пакет openssh-server командой:

sudo urpmi openssh-server

Bash
urpmi сам найдет и установит нужный пакет, включая все зависимости.

После установки:

Запуск SSH-сервера: Запустите SSH-сервер:

sudo systemctl start sshd

Bash
или

sudo service sshd start

Bash
Автозапуск при загрузке: Чтобы SSH-сервер автоматически запускался при загрузке системы, выполните:

sudo systemctl enable sshd

Bash
или

sudo service sshd enable
